## Getting Started

### Logging In

1. Navigate to the website URL provided by your administrator
2. You will see the login page
3. Enter your credentials:
   - **Username**: Your assigned username
   - **Password**: Your password
4. Click **"Sign In"**

> **Note**: If you forget your password, contact your administrator.

### First-Time Login

After logging in for the first time, you'll be taken to the **Dashboard** - your central hub for all activities.

## Dashboard Overview

The Dashboard provides a quick overview of your business performance.

### Dashboard Components

#### 1. **Financial Overview (This Month)**
- **Total Income**: Sum of all income transactions for the current month
- **Total Expenses**: Sum of all expense transactions for the current month
- **Net Profit**: Income minus Expenses

You can adjust the time period using the controls at the top to view different months.

#### 2. **Order Statistics**
- **Total Orders**: Number of orders in the selected period
- **Pending Orders**: Orders waiting to be processed
- **Delivered Orders**: Successfully completed orders
- **Orders Trend Chart**: Visual representation of order volume over time

### Navigation Menu

The navigation bar at the top allows you to access:
- **Dashboard** (Home icon) - Main overview
- **Orders** - Order management
- **Finances** - Income and expense tracking
- **Inventory** - Stock management
- **Settings** - System configuration

## Managing Orders

### Viewing Orders

1. Click **"Orders"** in the navigation menu
2. You'll see a table with all orders showing:
   - Order Number
   - Date
   - Customer (WhatsApp number & Emirate)
   - Product (Perfume choice & quantity)
   - Price
   - Status
   - Actions

### Searching Orders

Use the **search bar** at the top to find orders by:
- Order number
- Customer phone number
- Perfume name
- Emirate/location
- Order taker name
- Receipt number
- Coupon number
- Notes

The search happens automatically as you type.

### Filtering Orders

Click the **"Filter"** button to reveal additional filters:

| Filter | Description |
|--------|-------------|
| **Status** | Filter by: Pending, Processing, Shipped, Delivered, Cancelled |
| **Emirate** | Filter by UAE emirate location |
| **Order Taker** | Filter by staff member who took the order |

### Creating a New Order

1. Click the **"+ New Order"** button (top right)
2. Fill in the order form:

#### Customer Details
- **Date**: Order date (defaults to today)
- **WhatsApp Number**: Customer's WhatsApp contact (required)
- **Direct Phone**: Alternative phone number

#### Product Details
- **Perfume Choice**: Name/type of perfume ordered (required)
- **Quantity**: Number of items (minimum 1)
- **Price (AED)**: Total price of the order
- **Customization**:

#### Delivery Location
- **Emirate**: Select the UAE emirate or "Other" for international
- **Area**: Specific area within the emirate
- **Other Location**: For non-UAE deliveries

#### Additional Info
- **Receipt Number**: Payment receipt reference
- **Coupon Number**: Any discount coupon used
- **Order Status**: Current status of the order
- **Notes**: Any special instructions

3. Click **"Create Order"** to save

### Viewing Order Details

Click on any order row to see full details including:
- All order information
- Order timeline
- Edit and delete options

### Editing an Order

1. Click the **Edit** (pencil) icon in the Actions column
2. Or click on the order and then click "Edit"
3. Make your changes
4. Click **"Update Order"**

### Deleting an Order

1. Click the **Delete** (trash) icon in the Actions column

## Finances & Transactions

### Accessing Finances

1. Click **"Finances"** in the navigation menu
2. You may need to enter a separate finance password for security

### Finance Dashboard

#### Monthly Summary Cards
At the top, you'll see three cards showing for the **selected month**:
- **Total Income** (green): All money received
- **Total Expenses** (red): All money spent
- **Net Profit** (blue/gray): Income minus Expenses

Use the **month dropdown** to view different months.

### Adding a Transaction

Use the **"Add Transaction"** form:

1. **Type**: Select "Income" or "Expense"
2. **Amount**: Enter the amount in AED
3. **Category**: Choose a category:
   - Income: Sales, Services, Other Income
   - Expenses: Inventory, Salary, Marketing, Rent, Utilities, Other
4. **Description**: Brief note about the transaction
5. **Date**: When the transaction occurred
6. Click **"Add Transaction"**

### Viewing Transactions

Below the summary cards and form, you'll see a list of all transactions with:
- Date
- Type (Income/Expense)
- Category
- Description
- Amount

### Filtering Transactions

#### By Type
Click the tabs at the top:
- **All**: Show all transactions
- **Income**: Only income transactions
- **Expense**: Only expense transactions

#### By Search
Use the search bar to find transactions by category or description

#### By Date
- Use the **date preset dropdown** (Today, This Week, This Month, All Time)
- Or set custom date range using the date inputs

### Transaction Details

Click on any transaction row to see full details and access the delete option.

## Tips & Best Practices

### Order Management

1. **Always fill in the WhatsApp number correctly** - This is the primary customer contact
2. **Use notes for special requests** - Keep special instructions documented
3. **Update status regularly** - Keep customers informed by updating order status
4. **Record the price** - Helps with financial tracking

### Financial Tracking

1. **Record transactions daily** - Don't let them pile up
2. **Use clear descriptions** - "Sale - Oud Wood x2" is better than "Sale"
3. **Choose the right category** - Helps with expense analysis
4. **Review monthly** - Use the month selector to compare performance

### Security

1. **Log out when done** - Especially on shared computers
2. **Don't share your password** - Each user should have their own login
3. **Report suspicious activity** - Contact your administrator immediately

## Troubleshooting

### Common Issues

| Problem | Solution |
|---------|----------|
| Can't log in | Check username/password. Contact admin if locked out. |
| Page not loading | Refresh the page. Check internet connection. |
| Can't save order | Make sure all required fields (*) are filled |
| Transactions not showing | Check the date filter - it might be set to a specific period |
| Data looks old | Refresh the page to get latest data |

### Getting Help

If you encounter issues:
1. Try refreshing the page
2. Clear your browser cache
3. Try a different browser
4. Contact your system administrator
